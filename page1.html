Football Network Analysis

This project builds a network connecting Division I college football players, their colleges, and NFL teams. Edges in the network represent which college a player attended and which NFL team drafted them, with draft round encoded as edge weight. Network-based features were then extracted and used to train a machine learning model to predict which NFL team a player would be drafted to. This model had an accuracy of 0.79
